Don't wanna talk

when I'm on fire

Don't wanna talk

when I'm in pain

Don't wanna talk

when I'm tired

(and even if

I'm feeling sad)

Don't wanna talk

when I'm in bed

Don't wanna talk

during the sex

Don't wanna talk

during the breakfast

Don't wanna talk

and when I rest

Don't wanna talk

but you

don't stop.
